Introduction

This is basically a webpage that you can create your own background noise. As you see from the background image, sounds are related to night street of downtown and church on Christmas Eve. You can control each volumes by hovering on the panel icon at bottom-left side of the screen. To add some dynamics, I made a snowing effect. Also, fireworks and stars are generated when you click on the night sky. Finally, if you write something on the text panel below the main image and click ‘play letter’ button, the text appears on the night sky, line by line.

I do not believe in any religion. But I really like that romantic and warm atmosphere of Christmas. I tried to express and share this feeling I have about Christmas, to others. If you show a letter to your loved one under dark, warm lighting, it will give you a romantic experience with calm and sacred background music :D


Methods

Volume control panel

It’s kind of similar to what I’ve done for homework. I made the structure with html and css. And connected with javascript file ‘player3.js’ to play sounds automatically and control volumes. For sounds, I chose some sounds related to the background image. (firework, church bells, crowd, bonfire and Gregorian Chant) Gregorian chant is the ‘core’ sound for peaceful and romantic atmosphere.


Letter

It takes text input and splits by line, into an array. Then it adds element text of the array to a transparent canvas using ‘fillText’ method. I used an if statement to count the number of lines, and jquery to fade in and fade out the canvas element. Some people may not be diligent enough to write a letter by themselves. So I created a ‘Poem’ button. It automatically plays famous poems. (mostly related to winter and Christmas, and some are my favorites) I currently wrote seven poems and they are played in a random order.


Firework Link to reference

I got it from It uses jquery. Originally, this code enables you to customize many attributes(velocity, number of particles, …), but I fixed them to certain values make it simple. It still has some weak randomness about some attributes like launch speed, colors.


Stars Link to reference

As you see from the link, this is originally ‘random night sky gerenator’. I needed an effect that creates stars at the position of cursor, when you click on the mouse. I thought I could use and modify this reference for my purpose. The size of stars are random in the range that I set. Colors are also random between three colors, same as the original code.


Snowing Link to reference

I applied a snowing effect from the reference. It is a pure css3 file that simply moves three snow images according to time.
